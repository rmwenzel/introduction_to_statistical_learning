Exercise 2: Alternative roughness penalties for smoothing splines

We’re just going to describe the solutions instead of drawing them.

This is the smoothing spline discussed in the chapter – g^\hat{g}g^​ is the ordinary least squares line.

d.

Now the integral penalty forces g(3)=0g^{(3)} = 0g(3)=0, so ggg is necessarily polynomial of degree leqslant 2. In general, a quadratic has more freedom, so averaging over datasets, the quadratic will have smaller RSS, and hence g^\hat{g}g^​ will be quadratic.

e.

In this case, the m=3m = 3m=3 condition is irrelevant. Now we are minimizing the RSS over all functions ggg, so g^\hat{g}g^​ will be any function which has zero RSS (i.e. which has g^(xi)=yi\hat{g}(x_i) = y_ig^​(xi​)=yi​ for all iii). For example, the interpolation spline, or a step (piecewise constant) function which passes through the y_i will have zero RSS. Such a function isn’t unique.
