Assessing the Accuracy of the Model

Quality of fit (model accuracy) is commonly assessed using RSERSERSE and the R2R^2R2 statistic.

Residual Standard Errors

  • The RSE is a measure of the overall difference between the observed responses yiy_iyi​ and the predicted responses y^i\hat{y}_iy^​i​. Thus it provides a measure of lack-of-fit of the model – higher RSE indicates worse fit.

  • RSE is measured in units of YYY so it provides an absolute measure of lack of fit, which is sometimes difficult to interpret

The R-squared Statistic

  • The R2R^2R2 statistic is

R2=TSS−RSSTSS R^2 = \frac{TSS - RSS}{TSS}R2=TSSTSS−RSS​

where TSS=∑i(yi−y‾)2TSS = \sum_i (y_i - \overline{y})^2TSS=∑i​(yi​−y​)2 is the total sum of squares.

  • TSSTSSTSS measures the total variability in YYY, while RSSRSSRSS measures the variability left after modeling YYY by f(X)f(X)f(X). Thus, R2R^2R2 measures the proportion of variability in YYY that can be explained by the model. R2R^2R2 is dimensionless so it provides a good relative measure of lack-of-fit.

  • As R2→1R^2 \rightarrow 1R2→1, the model explains more of the variability in YYY. As R2→0R^2 \rightarrow 0R2→0, the model explains less 13. What constitutes a good R2R^2R2 value depends on context

  • We can also think of R2R^2R2 as a measure of the linear relationship between YYY and XXX. Another such measure is the correlation corr(X,Y)\text{corr}(X,Y)corr(X,Y), which is estimated by the sample correlation rrr. In the case of simple linear regression, R2=r2R^2 = r^2R2=r2.

Deciding on Important Variables

  • The task of finding which predictors are related to the response is sometimes known as variable selection.19

  • Various statistics can be used to judge the quality of models using different subsets of the predictors. Examples are Mallows CpC_pCp​ criterion, Akaike Information Criterion (AIC), Bayesian Information Criterion and adjusted R2R^2R2.

  • Since the number of distinct linear regression models grows exponentially with ppp exhaustive search is infeasible unless ppp is small. Common approaches to consider a smaller set of possible models are

    • Forward Selection Start with the null model M0M_0M0​ (an intercept but no predictors). Fit ppp simple regressions and add to the null model the one with lowest RSSRSSRSS, resulting in a new model M1M_1M1​. Iterate until a stopping rule is reached.
    • Backward Selection Start with a model MpM_pMp​ consisting of all predictors. Remove the variable with largest ppp-value, resulting in a new model Mp−1M_{p-1}Mp−1​. Iterate until a stopping rule is reached.
    • Mixed Selection Proceed with forward selection, but remove any predictors whose ppp-value is too large.

Model Fit

  • As in simple regression, RSERSERSE and R2R^2R2 are two common measures of model fit

  • In multiple regression, R2=Corr(Y,Y^)2R^2 = Corr(Y, \hat{Y})^2R2=Corr(Y,Y^)2, with the same interpretation as in simple regression. The model Y^\hat{Y}Y^ maximizes R2R^2R2 among all linear models.

  • R2R^2R2 increases monotonically in the number of predictors, but small increases indicate the low relative value of the corresponding predictor.

  • In multiple regression

RSS=RSSn−p−1 RSS = \sqrt{\frac{RSS}{n - p - 1}} RSS=n−p−1RSS​​

  • Visualization can be helpful in assessing model fit, e.g. by suggesting the inclusion of interaction terms

Predictions

There are 3 types of uncertainty associated with predicting YYY by Y^\hat{Y}Y^

  • Estimation Error. Y^=f^(X)\hat{Y} = \hat{f}(X)Y^=f^​(X) is only an estimate f(X)f(X)f(X). This error is reducible. We can compute confidence intervals to quantify it.

  • Model Bias. A linear form for f(X)f(X)f(X) may be inappropriate. This error is also reducible

  • Noise. The noise term ϵ\epsilonϵ is a random variable. This error is irreducible. We can compute predeiction intervals to quantify it.

Extensions of the Linear Model

The standard linear regression we have been discussing relies on the twin assumptions

  • Additivity: The effect of XiX_iXi​ on YYY is independent of the effect of XjX_jXj​ for j≠ij\neq ij​=i.
  • Linearity: YYY is linear in XiX_iXi​ for all iii.

We can extend the model by relaxing these assumptions

Removing the Additive Assumption

  • Dropping the assumption of additivity leads to the possible inclusion of interaction or synergy effects among predictors.

  • One way to model an interaction effect between predictors XiX_iXi​ and XjX_jXj​ is to include an interaction term, βi+jXiXj\beta_{i + j}X_iX_jβi+j​Xi​Xj​. The non-interaction terms βiXi\beta_i X_iβi​Xi​ model the main effects.

  • We can perform hypothesis tests as in the standard linear model to select important terms/variables. However, the hierarchical principle dictates that, if we include an interaction effect, we should include the corresponding main effects, even if the latter aren’t statistically significant.

Non-linear Relationships

  • Dropping the assumption of linearity leads to the possible includion of non-linear effects.

  • One common way to model non-linearity is to use polynomial regression 20, that is model f(X)f(X)f(X) with a polynomial in the predictors. For example in the case of a single predictor XXX Y=β0+β1X+⋯+βdXsY = \beta_0 + \beta_1 X + \dots + \beta_d X^s Y=β0​+β1​X+⋯+βd​Xs models YYY as a degree ddd polynomial in XXX

  • In general one can model a non-linear effect of predictors XiX_iXi​ by including a non-linear function of the XiX_iXi​ in the model

Potential Problems

Non-linearity of the Data

Residual plots are a useful way of vizualizing non-linearity. The presence of a discernible pattern may indicate a problem with the linearity of the model.

Correlation of Error Terms

  • Standard linear regression assumes Corr(ϵi,ϵj)=0\text{Corr}(\epsilon_i,\epsilon_j) = 0Corr(ϵi​,ϵj​)=0 for i≠ji\neq ji​=j.

  • Correlated error terms frequently occur in the context of time series.

  • Positively correlated error terms may display tracking behavior (adjacent residuals may have similar values).

Non-constant Variance of Error Terms

  • Standard linear regression assumes the variance of errors is constant across observations, i.e. V(ϵi)=σ2\mathbb{V}(\epsilon_i) = \sigma^2V(ϵi​)=σ2 for all 1⩽i⩽n1 \leqslant i \leqslant n1⩽i⩽n

  • Hetereoscedasticity, or variance which changes across observations can be identified by a funnel shape in the residual plot.

  • One way to reduce hetereoscedasticity is to transform YYY by a concave function such as log⁡Y\log YlogY or Y\sqrt{Y}Y​.

  • Another way to do this is weighted least squares. This weights terms in RSSRSSRSS with weights wiw_iwi​ inversely proportional to σi2\sigma_i^2σi2​ where σi2=V(ϵi)\sigma_i^2 = \mathbb{V}(\epsilon_i)σi2​=V(ϵi​).

Outliers

  • An outlier is an observation for which the value of yiy_iyi​ given xix_ixi​ is unusual, i.e. such that the squared-error (yi−y^i)2(y_i - \hat{y}_i)^2(yi​−y^​i​)2 is large

  • Outliers can have disproportionate effects on statistics e.g. R2R^2R2, which in turn affect the entire analysis (e.g. confidence intervals, hypothesis tests).

  • Residual plots can identify outliers. In practice, we plot studentized residuals

ϵ^ise^(ϵ^i)\frac{\hat{\epsilon}_i}{\hat{\mathbf{se}}(\hat{\epsilon}_i)} se^(ϵ^i​)ϵ^i​​

  • If an outlier is due to a data collection error it can be removed, but great care should be taken when doing this.

High Leverage Points

  • A high leverage point is a point with an unusual value of xix_ixi​.

  • High leverage points tend to have a sizable impact on f^\hat{f}f^​.

  • To quantify the leverage of xix_ixi​, we use the leverage statistic. In simple linear regression this is

hi=1n+(Xj−X‾)2∑j(Xj−X‾)2 h_i = \frac{1}{n} + \frac{(X_j - \overline{X})^2}{\sum_{j} (X_{j} - \overline{X})^2} hi​=n1​+∑j​(Xj​−X)2(Xj​−X)2​

Collinearity

  • Collinearity is a linear relationship among two or more predictors.

  • Collinearity reduces the accuracy of coefficient estimates 21

  • Collinearity reduces the power22 of the hypothesis test

  • Collinearity between two variables can be detected by the sample correlation matrix Σ^\hat{\Sigma}Σ^. A high value for ∣(Σ^)ij∣=∣corr(Xi,Xj)^∣|(\hat{\Sigma})_{ij}| = |\hat{\text{corr}(X_i, X_j)}|∣(Σ^)ij​∣=∣corr(Xi​,Xj​)^​∣ indicates high correlation between Xi,XjX_i, X_jXi​,Xj​ hence high collinearity in the data23.

  • Multicollinearity is a linear relationship among more than two predictors.

  • Multicollinearity can be detected using the variance inflation factor (VIF)24. VIF(β^i)=11−RXi∣X−i2 VIF(\hat{\beta}_i) = \frac{1}{1-R^2_{X_i|X_{-i}}}VIF(β^​i​)=1−RXi​∣X−i​2​1​ where RXi∣X−i2R^2_{X_i|X_{-i}}RXi​∣X−i​2​ is the R2R^2R2 from regression of XiX_iXi​ onto all other predictors.

  • One solution to the presence of collinearity is to drop one of the problematic variables, which is usually not an issue, since correlation among variables is seen as redundant.

  • Another solution is to combine the problematic variables into a single predictor (e.g. an average)

Comparison of Linear Regression and K-Nearest Neighbors

  • Linear regression is a parametric model for regression (with parameter β=(β0,…,βp)\beta = (\beta_0, \dots, \beta_p)β=(β0​,…,βp​)).

  • KNN regression is a popular non-parametric model, which estimates

f^(x0)=1K∑xi∈N0yi\hat{f}(x_0) = \frac{1}{K}\sum_{x_i\in\mathcal{N}_0} y_i f^​(x0​)=K1​xi​∈N0​∑​yi​

  • In general, a parametric model will outperform a non-parametric model if the parametric estimation f^\hat{f}f^​ is close to the true fff.

  • KNN regression suffers from the curse of dimensionality - as the dimension increases the data become sparse. Effectively this is a reduction in sample size, hence KNN performance commonly decreases as the dimension ppp increases.

  • In general parametric methods outperform non-parametric methods when there is a small number of observations per predictor.

  • Even if performance of KNN and linear regression is comparable, the latter may be favored for interpretability.
