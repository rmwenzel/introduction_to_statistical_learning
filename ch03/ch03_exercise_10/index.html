The statsmodel documentation has descriptions of these mean squared errors2.

  • The model mean squared error is “the explained sum of squares divided by the model degrees of freedom”
  • The residual mean squared error is “the sum of squared residuals divided by the residual degrees of freedom”
  • The total mean squared error is “the uncentered total sum of squares divided by n the number of observations”

In OLS multiple regression with nnn observations and ppp predictors, the number of model degrees of freedom are ppp, the number of residual degrees of freedom are n−p−1n - p - 1n−p−1, so we have

msemodel=1p(∑i=1n(y^i−y‾)2)\mathbf{mse}_{model} = \frac{1}{p}\left(\sum_{i = 1}^n (\hat{y}_i - \overline{y})^2\right)msemodel​=p1​(i=1∑n​(y^​i​−y​)2) mseresid=1n−p−1(∑i=1n(y^i−yi)2)\mathbf{mse}_{resid} = \frac{1}{n - p - 1}\left(\sum_{i = 1}^n (\hat{y}_i - y_i)^2\right)mseresid​=n−p−11​(i=1∑n​(y^​i​−yi​)2) msetotal=1n(∑i=1n(yi−y‾)2)\mathbf{mse}_{total} = \frac{1}{n}\left(\sum_{i = 1}^n (y_i - \overline{y})^2\right)msetotal​=n1​(i=1∑n​(yi​−y​)2)

From this we can draw some conclusions

  • It’s clear why msetotal\mathbf{mse}_{total}msetotal​ for the two models in this problem
  • The error mseresid\mathbf{mse}_{resid}mseresid​ is slightly smaller (0.25%0.25\%0.25%) for the second model, so it’s slightly at predicting the response.
  • The error mseresid\mathbf{mse}_{resid}mseresid​ is much larger (50%50\%50%) for the second model, so for this model the response predictions have a much greater deviation from average

From this perspective, the first model seems preferable, but it’s difficult to say. The tiebreaker should come from their performance on test data.
