\hypertarget{resampling-methods}{%
\section{Resampling Methods}\label{resampling-methods}}

\begin{center}\rule{0.5\linewidth}{\linethickness}\end{center}

    \begin{itemize}
\item
  \textbf{\emph{Resampling methods}} involve repeatedly drawing samples
  from a training set and refitting a model of interest on each sample
  in order to obtain additional information about the fitted model
\item
  Two of the most commonly used resampling methods are
  \textbf{\emph{cross-validation}} and the bootstrap
\item
  Resampling methods can be useful in \textbf{\emph{model assessment}},
  the process of evaluating a model's performance, or in
  \textbf{\emph{model selection}}, the process of selecting the proper
  level of flexibility.
\end{itemize}

    \hypertarget{cross-validation}{%
\subsection{Cross-validation}\label{cross-validation}}

    \hypertarget{the-validation-set-approach}{%
\subsubsection{The Validation Set
Approach}\label{the-validation-set-approach}}

    \begin{itemize}
\item
  Randomly divide the data into a \textbf{\emph{training set}} and
  \textbf{\emph{validation set}}. The model is fit on the training set
  and its prediction performance on the test set provides an estimate of
  overall performance.
\item
  In the case of a quantitative response, the prediction performance is
  measured by the mean-squared-error. The validation estimates the
  ``true'' \(\text{MSE}\) with the mean-squared error
  \(\text{MSE}_{validation}\) computed on the validation set.
\end{itemize}

    \hypertarget{advantages}{%
\subparagraph{Advantages}\label{advantages}}

    \begin{itemize}
\tightlist
\item
  conceptual simplicity
\item
  ease of implementation
\item
  low computational resources
\end{itemize}

    \hypertarget{disadvantages}{%
\subparagraph{Disadvantages}\label{disadvantages}}

    \begin{itemize}
\tightlist
\item
  the validation estimate is highly variable - it is highly dependent on
  the train/validation set split
\item
  since the model is trained on a subset of the dataset, it may tend to
  overestimate the test error rate if it was trained on the entire
  dataset
\end{itemize}

    \hypertarget{leave-one-out-cross-validation}{%
\subsubsection{Leave-One-Out Cross
Validation}\label{leave-one-out-cross-validation}}

    Given paired observations
\(\mathcal{D} = \{(x_1, y_1), \dots, (x_n, y_n)\}\), for each
\(1 \leqslant i \leqslant n\): - Divide the data \(\mathcal{D}\) into a
training set \(\mathcal{D}_{(i)} = \mathcal{D}\ \{(x_i, y_i)\}\) and a
validation set \(\{(x_i, y_i)\}\). - Train a model \(\mathcal{M}_i\) on
\(\mathcal{D}_{(i)}\) and use it to predict \(\hat{y}_i\). - The LOOCV
estimate for \(\text{MSE}_{test}\) is

\begin{verbatim}
$$CV_{(n)} = \frac{1}{n}\sum_{i=1}^n \text{MSE}_i$$
\end{verbatim}

where \(\text{MSE}_i = (y_i - \hat{y}_i)\)31

    \hypertarget{advantages}{%
\subparagraph{Advantages}\label{advantages}}

    \begin{itemize}
\tightlist
\item
  approximately unbiased
\item
  deterministic - doesn't depend on a random train/test split.
\item
  computationally fast in least squares regression
  \[CV_{(n)} = \frac{1}{n}\sum_{i=1}^n \left(\frac{y_i - \hat{y}_i}{1 - h_i}\right)^2\]
  where \(h_i\) is the Section \ref{high-leverage-points} of point i
\end{itemize}

    \hypertarget{disdvantages}{%
\subparagraph{Disdvantages}\label{disdvantages}}

    \begin{itemize}
\tightlist
\item
  Computationally expensive32 in general
\end{itemize}

    \hypertarget{k-fold-cross-validation}{%
\subsubsection{\texorpdfstring{\(k\)-fold
Cross-Validation}{k-fold Cross-Validation}}\label{k-fold-cross-validation}}

    Given paired observations
\(\mathcal{D} = \{(x_1, y_1), \dots, (x_n, y_n)\}\), divide the data
\(\mathcal{D}\) into \(K\) \textbf{\emph{folds}} (sets)
\(\mathcal{D}_1, \dots, \mathcal{D}_K\) of roughly equal size.33 Then
for each \(1 \leqslant k \leqslant K\):

\begin{itemize}
\item
  Train a model on \(\mathcal{M}_k\) on
  \(\cup_{j\neq k} \mathcal{D}_{j}\) and validate on \(\mathcal{D}_k\).
\item
  The \(k\)-fold CV estimate for \(\text{MSE}_{test}\) is

  \[CV_{(k)} = \frac{1}{k}\sum_{i=1}^k \text{MSE}_k\]

  where \(\text{MSE}_k\) is the mean-squared-error on the validation set
  \(\mathcal{D}_k\)
\end{itemize}

    \hypertarget{advantages}{%
\subparagraph{Advantages}\label{advantages}}

    \begin{itemize}
\tightlist
\item
  computationally faster than \(LOOCV\) if \(k > 1\)
\item
  less variance than validation set approach or LOOCV
\end{itemize}

    \hypertarget{disdvantages}{%
\subparagraph{Disdvantages}\label{disdvantages}}

    \begin{itemize}
\tightlist
\item
  more biased than LOOCV if \(k > 1\).
\end{itemize}

    \hypertarget{bias-variance-tradeoff-for-k-fold-cross-validation}{%
\subsubsection{\texorpdfstring{Bias-Variance Tradeoff for \(k\)-fold
Cross
Validation}{Bias-Variance Tradeoff for k-fold Cross Validation}}\label{bias-variance-tradeoff-for-k-fold-cross-validation}}

    As \(k \rightarrow n\), bias \(\downarrow\) but variance \(\uparrow\)

    \hypertarget{cross-validation-on-classification-problems}{%
\subsubsection{Cross-Validation on Classification
Problems}\label{cross-validation-on-classification-problems}}

    In the classification setting, we define the LOOCV estimate

\[CV_{(n)} = \frac{1}{n}\sum_{i=1}^n \text{Err}_i\]

where \(\text{Err}_i = I(y_i \neq \hat{y}_i)\). The \(k\)-fold CV and
validation error rates are defined analogously.

    \hypertarget{the-bootstrap}{%
\subsection{The Bootstrap}\label{the-bootstrap}}

    The bootstrap is a method for estimating the standard error of a
statistic34 or statistical learning process. In the case of an estimator
\(\hat{S}\) for a statistic \(S\) proceeds as follows:

Given a dataset \(\mathcal{D}\) with \(|\mathcal{D}=n|\), for
\(1 \leqslant i \leqslant B\): - Create a bootstrap dataset
\(\mathcal{D}^\ast_i\) by sampling uniformly \(n\) times from
\(\mathcal{D}\) - Calculate the statistic \(S\) on
\(\mathcal{D}^\ast_i\) to get a bootstrap estimate \(S^\ast_i\) of \(S\)

Then the bootstrap estimate for the \(\mathbf{se}(S)\) the sample
standard deviation of the boostrap estimates
\(S^\ast_1, \dots, S^\ast_B\):

\[\hat{se}(\hat{S}) = \sqrt{\frac{1}{B-1} \sum_{i = 1}^ B \left(S^\ast_i - \overline{S^\ast}\right)^2}\]
