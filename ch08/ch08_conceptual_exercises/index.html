Exercise 2: Why a boosted stump leads to an additive model

The final form of the boosting model (see algorithm 8.12) is

f^(X)=∑b=1Bλf^b(x)\hat{f}(X) = \sum_{b = 1}^B \lambda \hat{f}^b(x)f^​(X)=b=1∑B​λf^​b(x)

Since the model is a stump (d=1d=1d=1), for each bbb the estimate is a function of a single predcitor, that is f^b(X)=f^b(Xj)\hat{f}^b(X) = \hat{f}^b(X_j)f^​b(X)=f^​b(Xj​) for some jb∈{1,…,p}j_b \in \{1, \dots, p\}jb​∈{1,…,p}. We can then collect terms:

f^(X)=∑b=1Bλf^b(Xjb)=∑j=1pλf^j(Xj) \begin{aligned} \hat{f}(X) &= \sum_{b = 1}^B \lambda \hat{f}^b(X_{j_b})\\ &= \sum_{j = 1}^p \lambda \hat{f}_j(X_j) \end{aligned} f^​(X)​=b=1∑B​λf^​b(Xjb​​)=j=1∑p​λf^​j​(Xj​)​

where f^j(x)=∑λf^b(Xj)\hat{f}_j(x) = \sum \lambda \hat{f}^b(X_j)f^​j​(x)=∑λf^​b(Xj​) is a sum over all boosted trees which have XjX_jXj​ as a predictor.

This is the estimate of an additive model.

f(X)=∑j=1pfj(Xj)f(X) = \sum_{j=1}^p f_j(X_j)f(X)=j=1∑p​fj​(Xj​)

Exercise 3: Plotting classification error, Gini index, and entropy as functions of p^m1\hat{p}_{m1}p^​m1​ for the two class case

%matplotlib inline
import matplotlib.pyplot as plt
import numpy as np

def class_error(p_hat_m1):
    p_hat_m2 = 1 - p_hat_m1
    return 1 - p_hat_m2 if p_hat_m2 >= p_hat_m1 else 1 - p_hat_m1

def gini_index(p_hat_m1):
    p_hat_m2 = 1 - p_hat_m1
    return p_hat_m1 * ( 1 - p_hat_m1) + p_hat_m2 * ( 1 - p_hat_m2)

def entropy(p_hat_m1):
    p_hat_m2 = 1 - p_hat_m1
    if p_hat_m1 in {0, 1}:
        return 0
    else:
        return - np.sum(p_hat_m1 * np.log(p_hat_m1) + p_hat_m2 * np.log(p_hat_m2))

v_class_error = np.vectorize(class_error)
v_gini_index = np.vectorize(gini_index)
v_entropy = np.vectorize(entropy, otypes=[np.float])

x = np.linspace(0, 1, 100)
y_1, y_2, y_3  = v_class_error(x), v_gini_index(x), v_entropy(x)

fig, ax = plt.subplots(figsize=(15, 10))
ax.plot(x, y_1, '-b', label="Classification error")
ax.plot(x, y_2, '--r', label="Gini index")
ax.plot(x, y_3, ':g', label="Entropy")
